
# Minimal Next.js Equation Generator

Минималистичное приложение на Next.js, генерирующее рандомное уравнение,
его корни и решение по шаблону.

## Запуск
```
npm install
npm run dev
```

## Структура
- Генерация типа уравнения (1–5)
- Генерация корней
- Подбор коэффициентов
- Вывод уравнения, корней и решения
